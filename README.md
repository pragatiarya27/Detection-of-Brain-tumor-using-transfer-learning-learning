# Detection-of-Brain-tumor-using-transfer-learning-learning
Brain tumor detection using transfer learning applies pre-trained deep learning models to MRI scans to classify images as tumor or non-tumor. It improves accuracy and reduces training time by leveraging learned features from large datasets, enabling efficient and early diagnosis.
This project proposes a hybrid deep learning model that combines ResNet50, VGG19, and Vision Transformer (ViT) to classify brain tumors from MRI images. The fused CNN and transformer features improve accuracy by capturing both local patterns and global context.
Additionally, a U-Net–based segmentation model is used to identify tumor regions, making the system an effective AI-assisted tool for automated brain tumor detection and analysis.
